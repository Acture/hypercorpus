# MC Layer

`docs/mc/` stores short memory notes for working research decisions.

Use this layer for:

- venue strategy
- narrative tradeoffs
- one-way decisions that should be remembered across writing sessions
- paper-planning notes that are useful internally but should not be treated as polished paper text

Do not use this layer as the source of truth for:

- claim support
- implementation status
- bibliography

Those belong in `phase-decisions.md`, `current-implementation.md`, and `literature-map.tsv`.
